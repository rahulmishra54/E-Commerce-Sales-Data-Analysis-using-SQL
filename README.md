# E-Commerce Sales Data Analysis using SQL

## 📌 Overview
This project analyzes a real-world e-commerce dataset using SQL to extract meaningful business insights. It demonstrates SQL skills through data exploration, customer behavior analysis, sales trends, and performance reporting.

## 🚀 Features
- Sales and revenue analysis
- Customer purchase behavior analysis
- Product performance analysis
- Seller performance evaluation
- Order and delivery trend analysis
- Business insights using SQL queries

## 🛠️ SQL Concepts Used
- SELECT, WHERE, ORDER BY
- JOINs
- GROUP BY & HAVING
- Aggregate Functions
- Subqueries
- Common Table Expressions (CTEs)
- Window Functions
- Date Functions

## 📊 Business Insights
- Identified top-selling products and categories
- Analyzed monthly sales trends
- Evaluated customer purchasing patterns
- Measured seller performance
- Examined delivery times and order status

## 🧰 Tech Stack
- SQL
- MySQL

## 📂 Dataset
Real-world E-Commerce Sales Dataset (Olist)
